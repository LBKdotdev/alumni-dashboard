Input Focus Fix + Outlook Email Integration

What Shipped

Input Focus Persistence

Fixed a bug where typing in any input field (search, filters, outreach fields) lost focus after every keystroke. The full DOM re-render on state change was destroying and rebuilding inputs. Now captures the active element and cursor position before render and restores them after, so typing feels natural.

Outlook Web Email Integration

Replaced mailto: protocol (unreliable — requires system-level email handler config) with Outlook web compose deeplink. "Send Email" now opens a new tab to Outlook with To, Subject, and Body pre-filled. Works on any machine, any browser, no configuration needed.
