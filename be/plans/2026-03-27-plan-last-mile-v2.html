Ideas Bank (Phase 3+)

Ideas surfaced during the audit. Not committed to any timeline. Captured here so nothing gets lost.

Idea

Gmail API email integration

OAuth with Lisa's WesternU Gmail. "Send Email" creates a draft in her Gmail Drafts folder. She opens Gmail, reviews, clicks Send. Full HTML formatting, tracking pixels for open detection, reply threading. Replaces mailto:.

Idea

NaciLink branding

Name the product. Source a clean Vitruvian Man image (public domain, no Google watermark). Integrate brand into login gate, nav bar, favicon. Custom domain: nacilink.com. "The prototype" becomes a product with identity. Lake Nacimiento origin story for the name.

Idea

Alumni geographic map

Interactive map view of alumni locations. Pin clusters by state/city. Click a cluster to filter the directory. "168 alumni in Nevada" as a visual, not a number. Essential for conference planning. Could use Leaflet.js (lightweight, no API key) or Mapbox.

Idea

Smart search / natural language

"Mentors near Portland" or "class of 2010 donors who haven't been contacted." The data model supports complex queries. The search box currently only does text matching. An LLM-powered search layer would be transformative.

Idea

Lebanon campus data

COMP-Northwest (Lebanon, OR) currently shows "Coming Soon." Need registrar data or NPI batch query for Lebanon graduates. Estimated ~2,347 graduates since 2015. Two-campus comparison becomes real.

Idea

Outreach analytics dashboard

"You've reached 42 alumni this quarter. 28 opened. 12 replied. 3 became mentors." Tracks the full funnel from trigger detection to relationship outcome. Proves ROI to the Provost. Requires email tracking + persistence backend.

Idea

Calendar / timeline view

Visual calendar showing project deadlines, follow-up due dates, upcoming milestones, and trigger events on a timeline. "What's happening this week" as a view, not just the queue ranking.

Idea

Suggested actions that learn

Track which outreach drafts Lisa edits, which triggers she acts on first, which approaches get replies. Use that signal to rerank triggers and improve draft quality over time. The tool adapts to Lisa's instincts.

Idea

Multi-user with roles

Lisa (Dean) sees everything. Staff see their checklist items. Board members see the dashboard only. Cloudflare Access + Worker auth makes this possible without a heavy auth framework.

Idea

Real-time NPI + PubMed sync

Scheduled Worker that queries NPPES and PubMed weekly. Auto-generates triggers when changes are detected. No manual data refresh. The system watches for Lisa.

Idea

PDF export for profiles

"Print this profile for my meeting." One-page PDF with photo placeholder, engagement summary, touchpoint history, personal notes. Lisa brings it to a lunch meeting. Physical artifact from a digital tool.

Idea

Bulk outreach campaigns

Select 20 alumni from a project, generate personalized drafts for each (using templates + individual data), review all 20, send all 20. Mass personalization, not mass email. Each one reads like Lisa wrote it individually.
