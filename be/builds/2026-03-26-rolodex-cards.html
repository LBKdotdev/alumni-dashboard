~U: Rolodex Card Animation

3D flip-in, hover lift, page transition — the directory demands attention

What Was Updated

Rolodex Flip-In

Directory alumni cards now animate in from a tilted-back position (rotateX -40deg) like flipping forward on a Rolodex spindle. Staggered 50ms per card. CSS 3D perspective on the container at 1200px depth. Pure CSS transforms, 60fps, zero libraries.

Hover Lift

Hovering a card tilts it forward 2deg and lifts it 6px with a layered shadow (burgundy tint + ambient). Feels like pulling a card off the spindle to read it. Cubic-bezier easing for a natural, weighted feel.

Page Transition

Paginating (Next/Previous) triggers a flip-out animation first (cards tilt forward 30deg and fade), then the new page flips in. 200ms exit, then fresh flip-in. Smooth and intentional.
