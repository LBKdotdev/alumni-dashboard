Decisions Made

Color semantics: burgundy = action, gold = relationship

Profile accent bars follow a deliberate color code. Burgundy marks sections where Dr. Warren takes action (engagement, outreach, timeline). Gold marks sections about the person (notables, notes). Gray marks system data (provenance). This wasn't arbitrary — it maps to how Lisa actually uses the profile.

Tints stay under 6% opacity

Background tints are deliberately subtle — 4-6% opacity max. They're sensed more than seen. The goal is "I can tell these cards apart without thinking" not "every card is a different color." The border + tint work together; neither needs to be loud.

Notable lift uses gold, not generic shadow

Instead of a neutral gray shadow for elevated cards, the box-shadow uses gold-tinted rgba. This ties the elevation to the project's color language — gold = noteworthy — rather than just "this card is higher." VIP and notable alumni share the same lift because they're the same concept: people worth knowing.
