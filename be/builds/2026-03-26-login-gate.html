Authentication Logic

  • Access code: 2001 (hardcoded for prototype)
  • Session persistence via sessionStorage (survives refresh, clears on tab close)
  • Enter key submits (not just button click)
  • Invalid code: red flash on input, error message, auto-clear
  • App data doesn't load until authenticated (no network calls behind the gate)

Mobile Optimized

  • Max-width 380px card, responsive padding
  • 16px input font-size prevents iOS zoom on focus
  • Touch-friendly button sizing
  • Full viewport coverage on all screen sizes

Decisions Made

  • No username field. Single-user prototype, one code for everyone.
  • sessionStorage, not localStorage. Tab close = logged out. Security over convenience.
  • Data doesn't load until auth passes. The gate blocks everything, not just the UI.
  • Password input type hides the code from shoulder-surfing.
